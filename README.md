# How To Use
> ```cwifs.setup()``` — Initializes CWIFS

<br>

> ```cwifs.file.delete(filename)``` — Deletes "filename".

<br>

> ```cwifs.file.write(filename,content)``` — Write file "filename" with content "content".

<br>

> ```cwifs.file.read(filename)``` — Read file "filename" and return file contents as result.
